** Module flags
- +lsp ::   
  Enable LSP support for  ~typst-ts-mode~. 
  Requires [[doom-module::tools lsp]] and a language server
  (supports [[https://github.com/Myriad-Dreamin/tinymist][Tinymist]]).
- +tree-sitter ::
  Leverages tree-sitter for better syntax highlighting and structural text
  editing. Requires Emacs 29.1+ and [[doom-module::tools tree-sitter]].
  This is a must. There exist no ~typst-mode~.
